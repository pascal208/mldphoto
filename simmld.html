Pipeline de transmission :
Pour chaque pixel (R,G,B), on découpe chaque octet en deux nibbles de 4 bits. Chaque nibble est encodé via c = G·m mod 2 → 7 bits transmis. Sur le BSC, chaque bit est flippé avec probabilité p. Le décodeur calcule s = H·r mod 2 — si s ≠ 0, il pointe la position erronée et la corrige. Une erreur simple est toujours corrigible ; deux erreurs dans le même bloc produisent une correction vers la mauvaise position → pixel corrompu.

Capacité du code : distance de Hamming d = 3, soit t = 1 erreur corrigible par bloc de 7 bits. Rendement R = 4/7 ≈ 0.571.
